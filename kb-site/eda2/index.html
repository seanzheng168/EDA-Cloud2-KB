EDA Cloud 2.0 使用常見問題

1. 登入 EDA Cloud VPN 顯示 "無法建立工作階段"

請檢查目前電腦連線使用的實體IP資訊，並比對申請帳號時提供的IP是否相同，若目前使用IP與申請帳號IP不同則無法通過驗證登入，如需變更IP資訊請至EDA Cloud2.0帳號申請頁面提出 修改申請，帳號申請頁面>EDA Cloud/Service Lab帳號申請>選擇您的帳號"修改"，填寫正確資訊後送出即可。

查詢 IP 可參考：此連結

※EDA Cloud 2.0僅開放來源為學術網路IP連線，使用非學術網路IP連線將會導致登入驗證失敗。

※EDA Cloud 2.0 Service Lab(sa)帳號僅開放中心設置先進晶片實驗室(Service Lab)連線使用。

2. 登入 Citrix VPX 顯示 "無法登入請關閉您的瀏覽器"

此為 Cache 問題，請清除瀏覽器所有瀏覽資料並關閉 Citrix Workspace，再重新登入。

3. 點選虛擬桌面後閃退或顯示 "無法啟動"

1. 請清除瀏覽器快取、關閉 Citrix Workspace，重新登入測試。

2. 若無效，請移除 Citrix Workspace 並安裝 LTSR 版本。

3. 若 CPU 使用率長時間過高，請聯絡客服安排平台重啟。
